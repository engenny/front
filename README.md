# front

## Project setup
```
npm install
```

### ApexChart 
```
npm install apexcharts --save
```

### Vuex
```
npm install vuex --save
```

### Bootstrap Vue
```
npm i bootstrap-vue
```

### Bootstrap 
```
npm i bootstrap
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
